# Project Methodology

The project followed an end-to-end data analytics workflow.

## Phase 1

Data acquisition and understanding.

## Phase 2

Data cleaning and preprocessing.

## Phase 3

Database normalization.

## Phase 4

SQLite database creation.

## Phase 5

Exploratory SQL analysis.

## Phase 6

Advanced SQL analytics using:

* CTEs
* Window Functions
* Ranking Functions
* NTILE
* CASE Statements
* LAG
* Moving Averages

## Phase 7

Business insight generation and recommendations.

The methodology demonstrates a complete analytics workflow from raw data to executive reporting.
